# 느낀 것
- 깃헙액션과 ECR, ECS를 통한 CD/CD 파이프라인 구축이 기존에 내가 하던 방식과 어떤 차이점과 장점들이 있는지 알게 됐다. 
- 추가로 멘토링을 통해서 실무에서는 어떻게 쓰이고 있는지 고려해봐야할점은 무엇인지에 대해 듣고 나니 좀 더 많이 학습해야할 필요성을 느꼈다.

# 아쉬웠던 것
